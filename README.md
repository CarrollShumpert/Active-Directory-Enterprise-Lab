# Active Directory Domain Lab

## Project Overview
This lab simulates a small business network environment using Windows Server Active Directory. A domain controller was deployed to manage centralized authentication, and a Windows 11 workstation was joined to the domain.

## Technologies Used
- Windows Server
- Windows 11
- Active Directory Domain Services
- VirtualBox / VMware

## Lab Environment

| Machine | Role |
|-------|------|
| DC1 | Domain Controller |
| CLIENT1 | Windows 11 Workstation |

## Active Directory Structure

Departments created:

- HR
- IT
- Finance
- Sales

User accounts were created for each department and tested through domain login.

## Skills Demonstrated

- Active Directory setup
- User and department management
- Domain authentication
- Windows administration
- Troubleshooting DNS configuration
